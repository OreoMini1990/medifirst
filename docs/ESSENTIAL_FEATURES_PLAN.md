# MediFirst 필수 기능 개발 계획서

## 📋 문서 정보

- **작성일**: 2024년 12월
- **버전**: 1.0.0
- **대상**: 필수 기능 (Essential Features)
- **예상 개발 기간**: 2-3일

---

## 🎯 개발 목표

커뮤니티 기능의 핵심적인 필수 기능을 구현하여 사용자 경험을 개선하고 기본적인 게시판 기능을 완성합니다.

---

## 📝 필수 기능 목록

### 1. 직역별 커뮤니티 자동 필터링
**우선순위**: 최우선  
**예상 소요 시간**: 4-6시간

#### 목표
- 사용자가 가입 시 입력한 직업(role)에 따라 자동으로 해당 직역 게시판으로 진입
- 사용자는 직역 필터를 볼 필요 없음
- 해당 직역의 게시글만 표시
- 글쓰기 시 직역 선택 불필요 (자동으로 본인 직역으로 설정)

### 2. 조회수 기능
**우선순위**: 최우선  
**예상 소요 시간**: 3-4시간

#### 목표
- 게시글 조회수 추적 및 표시
- 중복 조회 방지 (같은 사용자가 같은 게시글을 여러 번 조회해도 1회만 카운트)
- 게시글 리스트 및 상세 페이지에 조회수 표시

## 🧪 테스트 계획

### 직역별 커뮤니티 자동 필터링 테스트
1. 로그인한 사용자가 직역별 커뮤니티 접근 시 본인 직역 게시글만 표시되는지 확인
2. 로그인하지 않은 사용자 접근 시 로그인 페이지로 리다이렉트되는지 확인
3. 직역 정보가 없는 사용자 접근 시 온보딩 페이지로 리다이렉트되는지 확인
4. 글쓰기 시 직역 선택 없이 자동으로 본인 직역이 설정되는지 확인
5. 다른 직역 게시글 상세 페이지 접근 시 차단되는지 확인

### 조회수 기능 테스트
1. 게시글 상세 페이지 진입 시 조회수가 증가하는지 확인
2. 같은 사용자가 같은 게시글을 여러 번 조회해도 1회만 카운트되는지 확인
3. 비로그인 사용자도 조회수가 증가하는지 확인
4. 게시글 리스트 및 상세 페이지에 조회수가 정확히 표시되는지 확인
