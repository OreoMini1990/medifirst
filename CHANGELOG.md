# Changelog

모든 주요 변경사항은 이 파일에 기록됩니다.

## [1.0.0] - 2024-12-03

### 추가됨 (Added)
- Next.js 15 프로젝트 초기 설정
- TypeScript + Tailwind CSS 통합
- shadcn/ui 컴포넌트 라이브러리 설정
- Supabase 클라이언트 및 서버 설정
- 인증 시스템 (이메일/비밀번호 로그인)
- 구글 OAuth 로그인 기능
- 프로필 자동 생성 기능
- 커뮤니티 페이지 (직역별, 자유게시판, 질문게시판)
- 개원·경영 페이지 (10개 카테고리 탭 UI)
- 구인·구직 페이지 기본 구조
- 반응형 레이아웃 및 네비게이션
- 데이터베이스 스키마 (profiles, posts, comments, jobs, resumes)
- Row Level Security (RLS) 정책 설정
- 미들웨어를 통한 인증 보호
- OAuth 콜백 처리 라우트

### 변경됨 (Changed)
- 기본 Next.js 템플릿을 MediFirst 프로젝트로 커스터마이징
- 레이아웃에 헤더 및 네비게이션 추가

### 문서화 (Documentation)
- README.md 작성
- 구글 OAuth 설정 가이드 작성
- 개발 완료 문서 작성
